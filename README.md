# SOCIB plots
Set of tools for plotting or processing data from the various facilities.
The files can be python or jupyter notebooks (formerly known as ipython notebooks). 

The files are organised by Facilities:
* HF Radar,
* Fixed stations,
* Gliders,
* ...

or by projects:
* Rissaga,
* CMEMS,
* Seaboards,
* ...
